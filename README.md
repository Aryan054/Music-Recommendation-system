# 🎵 Music Recommendation System

A Machine Learning-based Music Recommendation System that suggests songs to users based on their preferences. The project leverages data preprocessing, exploratory data analysis, and recommendation algorithms to provide personalized music recommendations.

## 📌 Features

* Recommend similar songs based on user-selected tracks.
* Content-based recommendation using song metadata.
* Fast and accurate recommendation generation.
* Data preprocessing and feature engineering.
* User-friendly interface for searching and recommending songs.
* Scalable architecture for adding new songs and recommendation techniques.

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Libraries:** Pandas, NumPy, Scikit-learn, Matplotlib
* **Machine Learning:** Content-Based Filtering
* **Notebook:** Jupyter Notebook
* **Version Control:** Git & GitHub

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/music-recommendation-system.git
```

2. Navigate to the project directory

```bash
cd music-recommendation-system
```

3. Create a virtual environment (optional)

```bash
python -m venv venv
```

4. Activate the virtual environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

5. Install dependencies

```bash
pip install -r requirements.txt
```

6. Run the application

```bash
python app.py
```

## 🚀 How It Works

1. Load the music dataset.
2. Clean and preprocess the data.
3. Extract meaningful song features.
4. Compute similarity between songs using machine learning techniques.
5. Accept a song name from the user.
6. Display the top recommended songs based on similarity.
